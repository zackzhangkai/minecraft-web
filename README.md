# Minecraft 风格网站

一个复古像素风格的 Minecraft 主题网页。

## 功能

- 🎒 物品栏 - 可点击交互的像素方块物品
- ⚒️ 合成台 - 3x3 像素合成格
- 👤 玩家 - Steve 角色展示
- 🧟 生物 - Creeper, 猪, 僵尸像素画

## 技术

- 纯 HTML/CSS/JS
- 像素艺术风格 (Press Start 2P 字体)
- CSS 绘制的 Minecraft 元素

## 运行

```bash
npx serve .
# 或
python3 -m http.server 8080
```

访问 http://localhost:8080
